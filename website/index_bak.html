三步上手

克隆仓库后按 README 本地构建；亦可根据团队流程打包分发。

克隆与安装依赖

克隆仓库 git clone https://github.com/acid1030/AxonClawX.git，进入目录后在 Node.js 环境下执行 npm install，准备 Electron 与原生模块构建。

开发调试

运行 npm run electron:dev 启动 Vite + Electron 联调。

打包发布

使用 npm run electron:build 及平台子命令产出安装包。

常见问题

若你与 ClawX 类桌面助手对比，可参考下列说明。

AxonClawX 与 OpenClaw / ClawX 是什么关系？

AxonClawX 是面向 OpenClaw 的 Electron 桌面客户端实现之一，聚焦配置管理、协作与监控；ClawX 等产品侧重研究型场景与渠道推送。二者可同属生态，定位不同。

是否必须会写代码？

日常使用以图形界面为主；高级用户可通过 JSON 与 Hooks 深度定制。开发与打包需要基本 Node.js 知识。

数据安全吗？

客户端本地运行，敏感配置与密钥由你保管；具体合规要求请结合内网策略与密钥管理实践评估。

支持哪些操作系统？

基于 Electron，可按 README 在 macOS、Windows、Linux 上开发与打包；具体安装包以你方 CI 产出为准。

和纯 OpenClaw CLI 相比优势是什么？

图形化配置中心、会话与代理管理、监控与告警一屏查看，降低运维与排障成本；CLI 仍可在本机并行使用。
