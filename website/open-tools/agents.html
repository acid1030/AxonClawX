Agent 与编排

多智能体协作、工具调用链路与可观测性，是生产级 AI 应用的常见底座。

  • LangChain — 链式调用、RAG、Agent 抽象与生态集成。
  • AutoGen — 微软多智能体对话与任务编排框架。
  • CrewAI — 角色化 Agent 团队与任务分工。
  • LlamaIndex — 数据连接器与索引，偏 RAG 与知识管线。
  • Langfuse — LLM 应用可观测、追踪与评估。
  • Swarm — OpenAI 轻量多 Agent 路由示例（教育向）。
