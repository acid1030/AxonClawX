自动化与集成

把 Webhook、数据库、HTTP 与模型 API 连成稳定管线，适合运营与内部工具。

  • n8n — 可视化工作流，社区节点丰富。
  • Flowise — 拖拽式 LLM 应用构建，易接向量库与工具。
  • smolagents — Hugging Face 轻量 Agent 库，强调简洁代码控制。
  • Apache Airflow — 数据工程与定时 DAG 编排事实标准之一。
  • Temporal — 有状态工作流与长事务，适合可靠任务编排。
  • PostgREST — 从 PostgreSQL 直接生成 REST API，快速数据面。

对外暴露工作流时请配置认证、速率限制与密钥管理。
