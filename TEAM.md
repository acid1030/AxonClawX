## 📋 任务分配规则

### 技能匹配

```
前端 UI → DASHBOARD-MVP / NODE-MANAGER-UI
依赖问题 → UI-FIX-DEPENDENCIES / DEPLOY-MASTER
WebSocket → WEBSOCKET-INTEGRATION
SSH/节点 → NODE-CONNECTOR-SERVICE
构建问题 → PROTON + DEPLOY-MASTER
质量审查 → SIGMA
任务调度 → ACE
```

### 超时机制

```
0-2min: 执行中 → 正常
2-5min: 无进展 → ACE 警告
5-10min: 仍无进展 → 重新分配
10min+: 多次失败 → 上报 Axon
```

### 验证流程

```
Agent 完成 → SIGMA 审查 → Proton 验证 → Axon 批准
   ↓           ↓            ↓           ↓
附证据     检查 DoD      实测运行    最终决策
```
