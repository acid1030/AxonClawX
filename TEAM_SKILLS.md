## 📋 技能激活方式

### 自动激活
当任务匹配技能描述时，Agent 自动读取对应 SKILL.md 并遵循指导。

### 手动激活
```
任务分配时指定技能：
"使用 kael-engineering 技能进行架构评审"
```

### 技能组合
复杂任务可激活多个技能：
```
前端任务：react-components + shadcn-ui + state-management
后端任务：api-design + kael-engineering + technical-design
测试任务：quality-testing + user-research + chromadb-memory
```
