# 🤖 Robots.txt 生成工具

**ACE Robots Utils** - 快速生成专业 robots.txt 文件

---

## 📦 功能特性

1. ✅ **Robots 规则定义** - User-agent, Allow, Disallow, Crawl-delay
2. ✅ **Sitemap 引用** - 支持多个 sitemap URL
3. ✅ **通配符支持** - `*` (任意字符), `$` (URL 结尾)
4. ✅ **预定义规则集** - 标准网站、SEO 工具、AI 爬虫等
5. ✅ **解析与合并** - 解析现有 robots.txt，合并多个配置

---

## 🚀 快速开始

### 基础使用

```typescript
import { generateRobotsTxt, createStandardWebRules } from './robots-utils-skill';

// 标准网站配置
const config = {
  rules: createStandardWebRules(),
  sitemap: ['https://example.com/sitemap.xml'],
  host: 'https://example.com',
};

const robotsTxt = generateRobotsTxt(config, { addComments: true });
console.log(robotsTxt);
```

**输出:**
```txt

Allow: /css
Allow: /js
Allow: /images
Disallow: /admin
Disallow: /private
Disallow: /api/private
Disallow: /.git
Disallow: /node_modules
Disallow: /search?

Sitemap: https://example.com/sitemap.xml
```

### 快捷构建函数

| 函数 | 描述 |
|------|------|
| `allowAll(userAgent)` | 允许所有访问 |
| `disallowAll(userAgent)` | 禁止所有访问 |
| `createStandardWebRules()` | 标准网站规则 |
| `createSearchEngineFriendlyRules()` | 搜索引擎友好规则 |
| `createApiServiceRules(prefix)` | API 服务规则 |
| `createCrawlerRestrictionRules(name, paths, delay)` | 爬虫限制规则 |

---

### 通配符工具

| 函数 | 描述 | 示例 |
|------|------|------|
| `exactMatch(path)` | 精确匹配 (使用 `$`) | `exactMatch('/login')` → `/login$` |
| `prefixMatch(path)` | 前缀匹配 (使用 `*`) | `prefixMatch('/api')` → `/api/*` |
| `containsMatch(pattern)` | 包含匹配 | `containsMatch('temp')` → `*temp*` |
| `extensionMatch(ext)` | 扩展名匹配 | `extensionMatch('pdf')` → `*.pdf$` |

---

### 预定义规则集

| 函数 | 阻止目标 |
|------|----------|
| `blockSeoTools()` | SEO 工具 (SEMrush, Ahrefs 等) |
| `blockAiCrawlers()` | AI 爬虫 (GPTBot, CCBot 等) |
| `blockScanners()` | 安全扫描器 |
| `blockAdBots()` | 广告爬虫 |

### 示例 3: 通配符高级用法

```typescript
import { 
  generateRobotsTxt,
  exactMatch,
  prefixMatch,
  containsMatch,
  extensionMatch
} from './robots-utils-skill';

const config = {
  rules: [
    {
      userAgent: '*',
      disallow: [
        exactMatch('/login'),           // 只匹配 /login
        prefixMatch('/api'),            // 匹配 /api/*
        containsMatch('temp'),          // 匹配包含 temp 的路径
        extensionMatch('pdf'),          // 匹配 *.pdf 文件
        extensionMatch('docx'),         // 匹配 *.docx 文件
      ],
    },
  ],
};

console.log(generateRobotsTxt(config));
```

**输出:**
```txt
User-agent: *
Disallow: /login$
Disallow: /api/*
Disallow: *temp*
Disallow: *.pdf$
Disallow: *.docx$
```

### 示例 5: 解析现有 robots.txt

```typescript
import { 
  parseRobotsTxt,
  mergeRobotsConfigs,
  generateRobotsTxt
} from './robots-utils-skill';

// 解析现有文件
const existing = parseRobotsTxt(`
User-agent: *
Disallow: /admin
Sitemap: https://old.com/sitemap.xml
`);

// 添加新规则
const newConfig = {
  rules: [
    ...existing.rules,
    {
      userAgent: 'GPTBot',
      disallow: ['/'],
    },
  ],
  sitemap: [...(existing.sitemap || []), 'https://new.com/sitemap.xml'],
};

console.log(generateRobotsTxt(newConfig));
```

### 示例 6: 合并多个配置

## ⚠️ 注意事项

1. **通配符语法**
   - `*` 可出现在任何位置，匹配任意字符
   - `$` 只能出现在末尾，匹配 URL 结尾

2. **规则优先级**
   - 更具体的 User-agent 优先于 `*`
   - Allow 和 Disallow 冲突时，更长路径优先

3. **Crawl-delay**
   - 并非所有爬虫都支持
   - Googlebot 忽略此指令

4. **文件大小**
   - 建议不超过 500KB
   - Google 只读取前 500KB
