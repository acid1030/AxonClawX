openclaw 报错 No API key found for provider "anthropic"，怎么解决？
14:22
🤖
这个错误说明 openclaw 找不到 Anthropic 的 API key。有两种解决方法：

方法一：命令行添加（推荐）
openclaw agents add main
运行后按提示输入你的 API key。

方法二：手动编辑配置文件
找到这个路径并编辑：/Users/t/.openclaw/agents/main/agent/auth-profiles.json
14:22
T
auth-profiles.json 的格式是什么样的？
14:23
🤖
格式如下：
json 
{
  "profiles": [
    {
      "provider": "anthropic",
      "api_key": "sk-ant-xxxxxxxxxxxxxxxx"
    }
  ]
}
把 sk-ant-... 换成你在 console.anthropic.com 生成的真实 key 即可。
